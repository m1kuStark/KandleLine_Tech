# KandleLine Tech

一个用于整理《日本蜡烛图技术》和常见 K 线形态的个人学习网站。

本项目仅用于学习和复盘，不构成投资建议。

## 使用方式

直接用浏览器打开 `index.html`。当前版本不需要安装依赖，也不需要启动服务。

## 颜色模式

页面支持两种 K 线颜色习惯：

- A 股：红涨绿跌
- 美股：绿涨红跌

颜色模式只影响图形展示，不改变形态定义。

## 收藏

形态标题旁的星标可以收藏当前形态。左侧“只看收藏”用于快速过滤平时最常用、最顺手的 K 线形态。

收藏保存在当前浏览器本地，不会提交到仓库，也不会影响其他使用者。

## 当前形态范围

当前版本已经覆盖单根 K 线、双根 K 线、三根组合、分型结构、窗口/跳空和持续形态。

新增形态时，优先在 `app.js` 的 `patterns` 数组中按现有模板添加：

- 名称、英文名、分类、倾向
- 出现场景、图形构成、市场含义
- 确认条件、失效条件、学习备注
- 标准 K 线数据和易混淆提醒

## 实战案例

每个 K 线形态都有“实战案例”区域。

- 本地上传：适合个人复盘，图片保存在当前浏览器本地，不会自动提交到仓库。
- 仓库协作：适合开源共建，把图片文件和案例记录一起提交到 Git 仓库。

推荐的仓库案例图片路径：

```text
cases/<pattern-id>/<stock-code>-<date>.png
```

推荐在 `cases.js` 中添加记录：

```js
window.KANDLELINE_REPO_CASES = [
  {
    id: "hammer-600519-20260531",
    patternId: "hammer",
    stock: "贵州茅台 600519",
    market: "A 股",
    date: "2026-05-31",
    outcome: "有效",
    image: "./cases/hammer/600519-20260531.png",
    note: "记录趋势背景、确认信号、失效位置和后续 3/5/10 个交易日表现。",
  },
];
```

建议同时收录有效案例和失败案例。失败案例通常更能帮助理解形态的适用边界。
